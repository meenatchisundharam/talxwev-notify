# Talxwev Notify

**Talxwev Notify** is a **lightweight, framework-agnostic toast notification library** for web applications. Works in **vanilla JS**, **Vue**, **React**, **Nuxt**, **Svelte**, or any environment that supports DOM.  

It provides **success, error, info, and warning notifications**, with auto-dismiss, pause-on-hover, and optional close buttons.  

---

## üöÄ Features

- No framework dependency ‚Äî works in any JS environment
- Easy to show toast notifications with just one function
- Auto-dismiss and sticky notifications
- Pause on hover
- Customizable styles and positions
- Full TypeScript support
- Supports CommonJS, ESM, and browser global (`TalxwevNotify`)

üíª Usage
Using ESM / CJS
js
Copy code
// Import the library
import TalxNotify from 'talxwev-notify';
// or
const TalxNotify = require('talxwev-notify').default;

// Show a simple notification
TalxNotify.notify("Hello world!");

// Show a success notification
TalxNotify.success("Data saved successfully!");

// Show an error notification
TalxNotify.error("Something went wrong!");

// Show an info notification
TalxNotify.info("This is an info message.");

// Show a warning notification
TalxNotify.warning("Be careful!");

// Dismiss a specific toast
const id = TalxNotify.notify("Closable toast");
TalxNotify.dismiss(id);

// Clear all notifications
TalxNotify.clear();

// Configure global default options
TalxNotify.configure({
  duration: 5000,
  position: "bottom-right",
  injectStyles: true
});
Using in Static HTML (Browser Global)
html
Copy code
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8" />
  <title>Talxwev Notify Test</title>
  <link rel="stylesheet" href="node_modules/talxwev-notify/dist/styles.css">
</head>
<body>
  <button id="successBtn">Show Success</button>
  <button id="errorBtn">Show Error</button>

  <script src="node_modules/talxwev-notify/dist/index.global.js"></script>
  <script>
    const successBtn = document.getElementById("successBtn");
    const errorBtn = document.getElementById("errorBtn");

    successBtn.addEventListener("click", () => {
      TalxwevNotify.success("This is a success toast!");
    });

    errorBtn.addEventListener("click", () => {
      TalxwevNotify.error("This is an error toast!");
    });
  </script>
</body>
</html>
‚öôÔ∏è Options
All notification options are available in notify() or global configure():

Option	Type	Default	Description
id	string	autogenerated	Unique ID for the toast
title	string	""	Title of the notification
message	string	""	Main content of the toast
type	string	"info"	`"info"
duration	number	4000	Auto-dismiss duration in ms (0 = sticky)
closable	boolean	true	Show close button
pauseOnHover	boolean	true	Pause auto-dismiss when hovered
position	string	"top-right"	`"top-right"
containerClass	string	""	Extra class for container
toastClass	string	""	Extra class for toast
injectStyles	boolean	true	Inject default CSS automatically
onClose	function	noop	Callback when toast is dismissed
render	function	null	Custom HTML render function

üí° Tips
Use TalxwevNotify.configure() to set global defaults for all notifications.

Use toastClass or containerClass to customize styling.

Can be used in SSR frameworks like Nuxt safely (notifications are no-op during server render).
